# Installing packages
```
sudo apt update
```
install node JS
```
sudo apt install nodejs
```
The installed version is displayed with the following command:
```
node -v
```
install mpm:
```
sudo apt install npm
```
Required modules:
```
npm install ejs
```
```
npm install express
```
```
npm install sequelize sqlite3
```
```
npm install express-session
```
